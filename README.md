# BookManager-
Projeto de Cadastro de Livros 
O BookManager é uma aplicação web desenvolvida em React para o cadastro, armazenamento e consulta de livros. O sistema permite que os usuários adicionem novos livros, visualizem os registros cadastrados e realizem pesquisas por obras utilizando a API OpenLibrary. Os dados cadastrados são armazenados localmente por meio do LocalStorage, garantindo que as informações permaneçam disponíveis mesmo após o fechamento do navegador. Durante o desenvolvimento foram utilizadas tecnologias como React, React Router DOM, JavaScript, HTML5, CSS3, Fetch API e Vite, aplicando conceitos de componentização, gerenciamento de estado, roteamento entre páginas e consumo de APIs. Entre os principais desafios enfrentados estiveram a integração com a API externa, a persistência dos dados, a validação dos formulários e a organização da aplicação em componentes reutilizáveis. O projeto foi desenvolvido com fins acadêmicos, proporcionando a aplicação prática dos conhecimentos de desenvolvimento Front-End e a criação de uma interface funcional, organizada e responsiva.
Instruções de Uso
1.	Instale as dependências do projeto utilizando o comando npm install. 
2.	Execute a aplicação com o comando npm run dev. 
3.	Acesse o endereço informado no terminal. 
4.	Utilize o menu de navegação para acessar as páginas do sistema. 
5.	Na página de cadastro, preencha os dados do livro e clique em Cadastrar. 
6.	Os livros cadastrados serão armazenados automaticamente no navegador. 
7.	Acesse a página de livros para visualizar os registros e realizar pesquisas na API OpenLibrary.
